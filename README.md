# Brazilian E-Commerce Business Analysis using SQL & Power BI

## Project Overview

This project presents an end-to-end business analysis of the Brazilian E-Commerce Public Dataset (Olist) using SQL and Power BI.

The objective of this project is to transform raw transactional data into meaningful business insights that support data-driven decision-making.

The analysis covers sales performance, customer behavior, delivery operations, product performance, and financial insights through SQL analysis and interactive Power BI dashboards.

---

## Project Objectives

- Analyze sales performance and revenue trends.
- Understand customer purchasing behavior.
- Evaluate delivery efficiency and customer satisfaction.
- Identify top-performing products and categories.
- Generate business insights to support strategic decision-making.
- Build an interactive Power BI dashboard.

## Key Business Insights

- Generated over 15 million in total revenue.
- Sales show seasonal purchasing patterns.
- A small number of product categories generate most revenue.
- Repeat customers spend more than one-time customers.
- Delivery delays reduce customer satisfaction.
- Certain cities and states generate significantly higher order volumes.
- Freight costs generally increase with product prices.

---

## Business Recommendations

- Improve customer retention strategies.
- Invest in high-performing product categories.
- Optimize delivery performance.
- Reduce shipping delays.
- Monitor KPIs using Power BI dashboards.
- Use seasonal demand patterns for inventory planning.
